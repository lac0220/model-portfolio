# Model portfolio

​A mobile-friendly web application designed for a professional model and artist. The website includes a dynamic gallery, blog section, and a fully functional contact form integrated with EmailJS for direct email communication.

## Features

- Responsive design (mobile-first approach)

- Image gallery with interactive slider

- Modal image zoom functionality

- Blog section

- Contact form with email integration

- Smooth client-side navigation

## Tech Stack

### Frontend:

- React.js

- SCSS

### Third-Party Libraries

- react-router-dom – client-side routing

- react-router-hash-link – smooth scrolling navigation

- react-hook-form – form validation

- emailjs/browser – email integration

- react-alice-carousel – image slider

- react-modal-image – image zoom modal

- node-sass – SCSS compilation

- font-awesome – icons
